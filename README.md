# utl_checkdigit
UTL_CHECKDIGIT provides procedures for generating and validating Luhn checkdigits in a number of bases
*) quotient - returns the result of integer division of the two numbers
*) Gen_Luhn_Check_Char - return the Luhn check digit for a number
*) Validate_Luhn_Check_Char - returns true if the right-most digit in a number is a valid Luhn check digit
*) Luhn_Check_Char_is_Valid - returns Y if the right-most digit in a number is a valid Luhn check digit
